# Sonny App Feature List

This is the running list of what the app currently does. We can keep updating this file whenever we add or change a feature.

## Collection Tracker

- Catalog-driven Sonny tracker powered by the guidebook/web data set.
- Search and browse Sonny figures with mapped artwork.
- Progress saved locally in the browser.

## Stock Shelf

- Add stock items from the tracker catalog with auto-filled name, series, and artwork.
- Set sell price, quantity, status, and trade-only mode for each stock card.
- Filter stock by search term and status.
- Sort stock by newest first, newest last, price, or name.
- Toggle between full stock cards and a shelf-style image view with four Sonnies per row on desktop.
- Mark an item sold and automatically create a Sonny Fund money-in entry.
- Move planned shipment items into stock.
- Persistent per-figure price history with inline history views on stock cards.

## Sonny Fund

- Track money-in and money-out transactions.
- Edit transaction name, date, type, and amount inline.
- See running balances and filtered summaries.
- Filter fund activity by the last 7 days, 30 days, 90 days, or lifetime.

## Shipments

- Create and save shipment plans.
- Track original shipment cost.
- Add shipment rows in bulk.
- Match shipment rows to exact Sonny catalog entries from a drop-down list.
- See a compact Sonny thumbnail and series label on each shipment row.
- Review previous in-stock prices for a Sonny while planning shipment pricing.
- Set sell price and keep value per shipment item.
- See shipment metrics for selling total, keeping total, cash loss, overall result, and unpriced count.
- Send shipment items into the stock shelf.

## Activity And Backups

- Timestamped activity log for stock, fund, shipment, and system actions.
- Search and filter the activity log by category.
- Download the activity log as CSV.
- Export and restore a full JSON backup of tracker, stock, fund, shipment, activity, and price-history data.
- Reset the active view to its seeded defaults.
